- VLA（Vision-Language-Action）模型、机器人控制（Robotics）、具身智能（Embodied AI）中的 agent
- 博弈论/计算经济学中的 agent（game-theoretic agent）
- 传统强化学习中与 LLM 无关的 agent
- 纯数学优化、信号处理等领域中借用 agent 概念的研究

评分标准 (1-10)，综合考虑「Agent 相关性」和「研究实质性」两个维度：

- 9-10: 核心 Agent 论文 + 有实质性技术贡献 + 有实验验证

  * Agent 记忆/规划/推理/工具使用的重要方法创新
  * 大规模 Agent 评测基准，含广泛模型对比和深入分析
  * 多智能体协作/通信的新范式，有实验验证

- 7-8: 高度相关 + 有贡献

  * Agent 安全/鲁棒性/沙箱/对抗攻防，有实现和评估
  * Agent 训练数据/奖励模型/critic 模型，有实验验证
  * Agent 用户模拟/交互评估，有对比实验

  * 通用 LLM 推理/对齐提升，可间接用于 Agent 但非 Agent 专门研究

  * 纯理论/形式化 Agent 论文，无任何实验或实现
  * 概念框架论文，仅有极小规模验证（如单个案例）

- 3-4: 弱相关（纯 NLP/CV 任务，仅间接相关）
- 1-2: 不相关
