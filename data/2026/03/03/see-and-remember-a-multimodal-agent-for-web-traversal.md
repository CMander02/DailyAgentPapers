# See and Remember: A Multimodal Agent for Web Traversal

## 原始摘要

Autonomous web navigation requires agents to perceive complex visual environments and maintain long-term context, yet current Large Language Model (LLM) based agents often struggle with spatial disorientation and navigation loops. In this paper, we propose generally applicable V-GEMS(Visual Grounding and Explicit Memory System), a robust multimodal agent architecture designed for precise and resilient web traversal. Our agent integrates visual grounding to resolve ambiguous interactive elements and introduces an explicit memory stack with state tracking. This dual mechanism allows the agent to maintain a structured map of its traversal path, enabling valid backtracking and preventing cyclical failures in deep navigation tasks. We also introduce an updatable dynamic benchmark to rigorously evaluate adaptability. Experiments show V-GEMS significantly dominates the WebWalker baseline, achieving a substantial 28.7% performance gain. Code is available at https://github.com/Vaultttttttttttt/V-GEMS.

## Q&A 论文解读

### Q1: 这篇论文试图解决什么问题？

这篇论文旨在解决基于大语言模型（LLM）的智能体在自主网页导航任务中存在的核心缺陷。研究背景是，随着互联网成为最大的动态知识库，能够像人类一样直接与图形用户界面（GUI）交互的自主网页导航智能体，对于AI训练、深度研究和自动化工作流至关重要。视觉语言模型（VLMs）的引入使智能体能够感知布局语义和复杂UI元素，超越了仅依赖文本DOM的传统方法。

然而，现有方法（如论文提及的WebWalker基线）仍存在明显不足。首先，**模态效率低下**：主要依赖文本DOM或标准LLM推理，难以准确解析视觉密集或模糊的交互元素（如区分可点击的横幅与普通图片），导致误操作或错过交互机会。其次，**导航停滞**：智能体缺乏明确的内部状态追踪，在复杂的深层页面结构中容易迷失方向，陷入导航循环或无法有效回溯，如同没有“后退”按钮的浏览器。最后，**符号不准确**：通用LLM在需要精确量化推理的任务（例如“查找恰好5篇论文”）中容易产生算术幻觉和聚合错误。

因此，本文要解决的核心问题是：如何构建一个**鲁棒且精确**的多模态网页遍历智能体，以同时克服**视觉歧义、长程上下文丢失导致的导航迷失以及符号推理不准确**这三重挑战。为此，论文提出了V-GEMS框架，通过集成自适应视觉感知、显式记忆栈和符号化工具，旨在实现精准的交互元素定位、结构化的路径记忆与回溯，以及可靠的数值处理，从而提升智能体在复杂动态网页环境中长期导航的准确性与稳定性。

### Q2: 有哪些相关研究？

本文的相关研究主要可分为三类：方法类、应用类与评测类。

在**方法类**研究中，早期工作主要依赖基于规则或行为克隆的技术，通过解析DOM结构（如XPath）实现网页自动化，但泛化能力有限。随着深度学习发展，研究转向利用语言模型，特别是近期多模态融合成为解决视觉模糊性的关键方案，例如引入视觉语言模型（VLM）处理图像按钮等元素。本文在此基础上，进一步优化智能体的内部认知过程，通过LLM与VLM的协同切换提升效率与可靠性。现有网页导航智能体可分为两类：一是训练专用小模型以提升特定导航能力；二是主流方法基于LLM（如ReAct框架）进行任务规划，但常存在算术幻觉和状态管理缺失问题。本文通过引入显式符号计数器和状态化URL栈，直接解决了这些缺陷，使智能体从被动规划转变为具备记忆的探索者。

在**应用类**研究中，网页遍历智能体常面临空间迷失和导航循环等问题，本文提出的V-GEMS架构通过视觉定位和显式记忆系统，增强了复杂视觉环境感知与长程上下文保持能力，显著提升了导航精度与鲁棒性。

在**评测类**方面，网页遍历基准从简单爬虫场景发展到复杂指令跟随任务，如Mind2Web、MMInA等。然而，静态数据集存在“时效性鸿沟”，易因网页动态变化而过时。本文创新性地提出了可更新的动态评估流程（EverWebQA基准），确保评测能持续反映真实网页环境，克服了以往依赖固定快照导致的复现性限制。

### Q3: 论文如何解决这个问题？

论文通过提出V-GEMS（视觉定位与显式记忆系统）这一多模态智能体架构，系统性地解决了自主网页导航中存在的空间迷失和循环导航问题。其核心方法围绕一个增强的双智能体（探索者与评判者）框架展开，并集成了一个专门设计的工具套件，以应对记忆、计算和感知方面的瓶颈。

整体框架包含三个关键组件：符号计数器、URL栈和US计算器。首先，为消除算术幻觉，系统引入了**符号计数器模块**。它作为确定性的逻辑护栏，将定量追踪与语言推理解耦。该计数器根据用户查询解析约束，激活两种执行模式之一：在目标明确的配额搜索中，它充当智能进度条，通过语义去重过滤验证实体唯一性，并在达到目标阈值时强制提前退出循环；在开放式统计任务中，它则切换为覆盖度监控角色，驱动探索者遍历完整的分页链直至环境穷尽，确保统计结果基于全面观察而非估计。

其次，为解决导航停滞和循环问题，系统设计了一个**URL栈模块**，作为外部拓扑记忆，实施稳健的深度优先搜索策略。该模块通过两个原子操作管理状态转换：当探索者决定遍历超链接到子节点时，执行“推送”操作，对当前URL进行快照，形成有序的探索路径；当渲染新页面后，若检测到无效或无关内容，探索者立即自主执行“弹出”操作，将环境回滚到上一个有效状态。这种基于栈的状态操作形式化，有效防止了循环并保证了系统能从局部最优回溯以探索替代分支。

最后，为平衡成本与性能，系统采用了**US计算器模块**，用于动态评估页面的文本可解性，从而自适应地触发视觉语言模型辅助。该模块根据学习到的阈值τ，通过一个综合评分函数Φ(s_t)来决定模态切换策略。该评分聚合了文本质量、语义相关性、结构清晰度和特殊案例惩罚四个启发式维度，旨在捕捉基于文本的DOM表示的具体失败模式。若评分低于阈值，评判者智能体则判定文本表征不足以反映实际页面，从而触发VLM流程以确保决策的准确性。

创新点主要体现在：1）通过外部化、模块化的工具（计数器、URL栈）显式管理长期状态与算术逻辑，增强了系统的确定性与鲁棒性；2）引入基于评分的自适应模态切换机制，在保证感知精度的同时优化了计算效率；3）将网页导航形式化为可回溯的图遍历过程，并通过栈结构实现了系统的状态化回溯能力，从根本上避免了导航循环。

### Q4: 论文做了哪些实验？

论文实验主要包括三部分：实验设置与基准构建、与基线方法的对比、以及模块消融分析。

**实验设置与数据集**：研究构建了动态基准测试EverWebQA，以应对网页内容随时间变化的问题。该数据集通过模拟人类导航行为（从简单检索到复杂信息合成）生成，包含680个双语（中英文）问答对，覆盖教育、会议、组织和游戏四个领域。问题分为单源（从根页面线性导航至目标深度页面）和多源（需整合同一网站内不同页面的信息）两类，并依据导航深度定义了易、中、难三个难度等级。

**对比方法与主要结果**：研究以广泛认可的先进框架WebWalker作为基线进行零样本对比。实验采用Qwen-3-Coder-Plus作为基础语言模型，Qwen3-VL-Plus作为视觉语言模型。主要结果显示，V-GEMS在平均成功率上显著优于WebWalker基线，从0.49提升至0.65，相对性能增益达到28.7%。具体而言，V-GEMS在复杂的多源硬任务和布局不规则的领域（如会议和游戏）中表现尤为突出，分别验证了其显式记忆栈在跨页面状态保持和视觉模块在复杂元素感知上的优势。

**关键数据指标**：
*   **整体性能**：平均成功率0.65（基线0.49），相对提升28.7%。
*   **领域性能提升**：在会议领域提升33.33%，在游戏领域提升56.25%。
*   **消融实验**：通过Shapley值量化模块贡献，例如在单源中等任务中视觉模块贡献值接近100，在多源简单任务中URL栈贡献值达到90，揭示了不同模块在感知与导航任务中的专门化作用。

### Q5: 有什么可以进一步探索的点？

基于论文的局限性，未来研究可从以下方向深入探索。首先，在计算效率方面，可探索更高效的视觉-语言模型融合架构，例如设计轻量级视觉编码器或采用动态计算分配策略，在保证感知精度的同时降低延迟。其次，针对开放世界导航，需研究从无边界起点（如搜索引擎）自主规划任务的能力，这可能涉及结合网络知识图谱与分层目标分解技术。再者，引入强化学习实现自我优化是关键方向，通过环境交互反馈迭代策略，以提升在复杂路径中的全局最优性。此外，论文未涉及跨网站泛化与安全伦理考量，未来可探索领域自适应机制以应对多样化网站结构，并设计安全约束防止恶意导航行为。最后，可考虑将显式记忆系统扩展为可学习的神经记忆网络，以更灵活地捕获和利用长期时空上下文。

### Q6: 总结一下论文的主要内容

该论文针对基于大语言模型（LLM）的智能体在自主网页导航任务中存在的空间迷失和导航循环问题，提出了一种名为V-GEMS的多模态智能体架构。其核心贡献在于通过视觉定位解决交互元素的模糊性，并引入一个具有状态跟踪功能的显式记忆栈，使智能体能够结构化记录其遍历路径，从而实现有效回溯并防止在深度导航任务中出现循环失败。方法上，V-GEMS通过符号计数器、URL栈和自适应URL计算器等模块，将特定认知功能与随机生成过程解耦，增强了智能体的可靠性和精确性。实验在一个可更新的动态基准上进行，结果表明V-GEMS显著优于WebWalker基线，性能提升了28.7%。论文结论指出，这种解耦设计对于构建可靠、高效的网页智能体至关重要，并展望了未来在提升效率、实现轻量化模型以及通过强化学习优化长程轨迹等方面的研究方向。
