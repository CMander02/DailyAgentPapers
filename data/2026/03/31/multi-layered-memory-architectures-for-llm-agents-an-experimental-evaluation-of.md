# Multi-Layered Memory Architectures for LLM Agents: An Experimental Evaluation of Long-Term Context Retention

## 原始摘要

Long-horizon dialogue systems suffer from semanticdrift and unstable memory retention across extended sessions. This paper presents a Multi-Layer Memory Framework that decomposes dialogue history into working, episodic, and semantic layers with adaptive retrieval gating and retention regularization. The architecture controls cross-session drift while maintaining bounded context growth and computational efficiency. Experiments on LOCOMO, LOCCO, and LoCoMo show improved performance, achieving 46.85 Success Rate, 0.618 overall F1 with 0.594 multi-hop F1, and 56.90% six-period retention while reducing false memory rate to 5.1% and context usage to 58.40%. Results confirm enhanced long-term retention and reasoning stability under constrained context budgets.

## Q&A 论文解读

### Q1: 这篇论文试图解决什么问题？

这篇论文旨在解决大语言模型（LLM）智能体在长程多轮对话中面临的核心挑战：如何在有限的计算资源（即受约束的上下文窗口）下，实现长期、稳定的上下文信息保留。研究背景是，尽管LLM在对话和任务推理方面表现出色，但其固定的上下文长度和未经控制的内存累积，导致随着对话轮次增加，早期关键信息被压缩或丢弃，从而引发语义漂移（如人物设定不一致、实体指代漂移）、事实不稳定以及记忆保持力下降等问题。现有方法存在明显不足：分层工作内存方法主要关注单会话内任务，缺乏跨会话的持久性；基于检索的架构虽提升了相关性选择，但容易积累错误记忆；基于参数的记忆方法能测量保留情况，却无法对语义稳定性进行显式结构控制；上下文压缩技术减少了令牌使用量，但未能维持跨会话的持久多层记忆。总之，现有方案往往顾此失彼，要么提升了效率但未稳定记忆保留，要么改善了召回但未控制记忆漂移。因此，本文要解决的核心问题是：设计一种结构化的多层记忆架构，在有限的上下文预算内，实现对多会话对话序列的长期上下文保持，同时维持计算效率，避免二次方的上下文增长。具体而言，论文提出了一个包含工作记忆、情景记忆和语义记忆的多层记忆框架，并辅以自适应检索门控和保留正则化机制，以期在控制跨会话语义漂移的同时，提升长期保留能力和推理稳定性。

### Q2: 有哪些相关研究？

本文的研究与多个方向的相关工作紧密关联，主要可分为**记忆架构设计**、**上下文管理与压缩**以及**评测基准**三大类。

在**记忆架构设计**方面，相关工作普遍采用分层或分层的记忆系统以管理长期信息。例如，Hu等人开发了分层工作记忆机制，将子目标组织为紧凑的记忆块；Phadke等人构建了包含工作、总结、归档和标记层的“真相维护”记忆代理以控制幻觉；Wadkar定义了一个由短期、情景、语义和程序性记忆组成的四层代理。本文提出的多层记忆框架（工作、情景、语义层）与这些工作同属一类，其核心区别在于集成了**自适应检索门控和保留正则化**机制，旨在更系统地控制跨会话的语义漂移，并在保持计算效率的同时实现有界的上下文增长。

在**上下文管理与压缩**方面，相关研究致力于解决长上下文带来的计算负担。例如，Shen等人提出了分层KV淘汰策略以加速长上下文解码；Wang等人引入了交叉注意力压缩模型来凝练上下文。本文的工作重点不在于压缩算法本身，而是通过其记忆架构设计，在源头（即记忆的存储与检索层面）实现对上下文长度的有效控制，从而与这些压缩技术形成互补。

在**评测基准**方面，Maharana等人构建的LOCOMO基准以及LOCCO、LoCoMo等数据集，专门用于评估模型在扩展对话中的长期记忆保持能力。本文的实验正是在这些基准上进行的，其改进的性能指标（如成功率、F1值和记忆保留率）直接证明了所提框架相对于先前方法（如Kang等人的三层内存操作系统或Shah等人的自适应三层记忆结构）的有效性。

### Q3: 论文如何解决这个问题？

论文通过提出一个结构化的多层记忆框架来解决长程对话中的语义漂移和记忆不稳定问题。核心方法是将对话历史分解为工作记忆、情景记忆和语义记忆三个层次，并辅以自适应检索门控和保留正则化机制，从而在有限上下文约束下实现长期记忆的稳定保持。

整体框架以对话会话序列为输入，每个会话经过编码后分别更新三层记忆状态。工作记忆通过有界编码函数仅保留最近k个话语的令牌级信息，确保计算效率与局部相关性。情景记忆采用递归累积方式，将当前会话摘要与上一会话记忆按衰减参数α混合，平衡细节保留与抽象程度。语义记忆则通过抽象函数将情景记忆映射为结构化的实体-事件图，提炼稳定的人格与事件结构，抑制冗余。

关键技术包括自适应检索门控和保留正则化。检索时，系统根据当前查询与各层记忆的语义相似度动态计算权重γ_i，通过softmax函数（由β控制锐度）实现层间优先级分配，从而减少无关语义漂移并支持多跳推理。记忆融合阶段通过交叉注意力机制整合短期输入与长期记忆信号。为控制语义漂移，论文引入保留正则化损失项，约束相邻会话间语义记忆嵌入的L2范数变化，惩罚突变的结构偏移，与生成损失共同优化。

创新点主要体现在：1）分层记忆架构明确分离短期、中期与长期信息，替代原始历史拼接，从根本上限制上下文增长；2）递归衰减与图结构抽象实现了记忆的渐进式压缩与稳定化；3）自适应检索机制根据上下文动态调整记忆层贡献，提升推理稳定性；4）保留正则化将记忆一致性作为显式优化目标，有效缓解灾难性遗忘。这些设计共同保障了在受限上下文预算下长期保持与推理性能的平衡。

### Q4: 论文做了哪些实验？

论文在三个长程对话基准测试上进行了实验：LOCOMO（主要评估人设一致性和长期记忆连续性，平均588.2轮对话和27.2个会话）、LOCCO（量化跨控制时间间隔的记忆衰减和多个周期后的保留稳定性）以及LoCoMo（评估跨实体、多跳、时序、因果和开放域类别的结构化推理性能）。实验设置采用统一的解码配置，控制上下文窗口边界和保留衰减因子，并调整超参数α、β和λ以稳定语义漂移而不增加计算开销。

对比方法包括分层工作记忆、记忆操作系统和基于保留的参数记忆基线。评估指标涵盖成功率（SR）、F1分数、BLEU-1、多周期后保留率和上下文利用率。

主要结果如下：在LOCOMO上，MLMF的成功率达到46.85（基线42.00），整体F1为0.618（基线0.583），多跳F1为0.594（基线0.550），BLEU-1为0.632（基线0.599）。在LOCCO上，六周期后保留率达到56.90%（基线48.25%），错误记忆率降至5.1%（基线6.8%），上下文使用率降至58.40%（基线64.98%），解码吞吐量达到10.4倍加速（基线9倍）。消融实验表明，移除语义层导致保留率最大下降（至50.84%），而移除保留正则化则使错误记忆率升至6.9%。所有改进在配对t检验下均具有统计显著性（p < 0.01）。

### Q5: 有什么可以进一步探索的点？

该论文提出的多层记忆架构在控制语义漂移和提升长期记忆保持方面取得了进展，但仍存在一些局限性和可探索的方向。首先，实验主要基于特定数据集（如LOCOMO等），其任务类型和对话复杂度可能有限，未来需要在更开放、动态的真实场景中验证泛化能力。其次，框架中的记忆分层与门控机制虽有效，但参数设计（如正则化强度、检索阈值）可能依赖经验调整，可探索自适应学习这些超参数的方法，例如通过元学习或强化学习动态优化。此外，当前架构侧重于文本对话，未来可扩展至多模态交互（如视觉、语音），研究跨模态信息的长期记忆整合。另一个方向是增强记忆的主动遗忘与重点强化机制，模仿人类记忆的选择性保留，以进一步提升效率。最后，可结合神经符号方法，在语义层引入结构化知识表示，以支持更复杂的推理和常识迁移。

### Q6: 总结一下论文的主要内容

这篇论文针对长程对话系统中存在的语义漂移和记忆不稳定问题，提出了一个多层记忆框架。该框架将对话历史分解为工作记忆、情景记忆和语义记忆三个层次，并引入了自适应检索门控和记忆保留正则化机制。其核心贡献在于通过这种结构化的记忆架构，在控制跨会话语义漂移的同时，实现了有界的上下文增长并保持了计算效率。实验在LOCOMO、LOCCO和LoCoMo数据集上进行，结果表明该方法显著提升了性能：取得了46.85%的成功率、0.618的总体F1值（其中多跳推理F1为0.594）以及56.90%的六周期记忆保留率，同时将错误记忆率降低至5.1%，并将上下文使用量减少到58.40%。主要结论证实，该框架在受限的上下文预算下，有效增强了智能体的长期记忆保留和推理稳定性。
